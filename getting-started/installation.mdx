## MCP Server (for AI coding agents)

The fastest way to get started. Run the wizard from your terminal and it will authenticate you, detect your installed coding agents, and configure the BackAnt MCP server automatically:

```bash
npx backant
```

The wizard opens a browser for OAuth login, then writes the MCP configuration to each detected agent. No manual config needed.

<Tip>
  Supports 30+ agents including Claude Code, Cursor, Windsurf, VS Code, Copilot, Cline, and more. See [Agentic Development](/getting-started/agentic-development) for the full list.
</Tip>

## CLI (for your terminal)

The `ant` CLI scaffolds and extends Flask projects from the command line.

### Requirements

- Python 3.11+
- Docker & Docker Compose (to run generated projects)

### Install via pip

```bash
pip install backant-cli
```

<Tip>Use a virtual environment to keep your global Python environment clean.</Tip>

### Verify the installation

```bash
ant --help
```

Expected output:

```
Usage: ant [OPTIONS] COMMAND [ARGS]...

Options:
  --report  Output machine-readable JSON instead of human-readable text.
  --help    Show this message and exit.

Commands:
  generate  Generate API projects, routes, and subroutes.
```
